# CLI Overview

The FARM CLI enables project creation and management through a unified command interface inspired by Next.js. It supports comprehensive arguments with smart prompts so developers can quickly scaffold projects or provide exact options.

## Project Creation

Key options from the `farm create` command include:

```bash
farm create <project-name> [options]
--template basic|ai-chat|ai-dashboard|ecommerce|cms|api-only|custom
--features auth,ai,realtime,payments,email,storage,search,analytics
--database mongodb|postgresql|mysql|sqlite
```

The CLI can run interactively when flags are omitted or operate non-interactively when all options are supplied.

## Example Scenarios

```bash
# Interactive creation
farm create my-app

# Template and features specified
farm create my-app --template ai-chat --features auth,ai
```

These flows mirror the `CLI Command Structure & Flow` sketch in the architecture documents.
