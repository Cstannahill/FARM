# Changelog

All notable changes to FARM Framework will be documented in this page.

## [2.0.0] - 2025-06-01

### Added

- 🚀 **New Authentication System** - Built-in OAuth2, JWT, and session management
- 📊 **Advanced MongoDB ODM** - Improved query builder and relationship handling
- 🔧 **CLI Enhancements** - New commands for project scaffolding and migration
- 🌐 **Real-time WebSocket Support** - Built-in WebSocket server with React hooks
- 📱 **Mobile-First Templates** - Responsive templates optimized for mobile devices
- 🧪 **Testing Utilities** - Comprehensive testing helpers and fixtures

### Improved

- ⚡ **40% faster build times** - Optimized Vite configuration and bundling
- 🎯 **Enhanced TypeScript support** - Better type inference and autocompletion
- 🛡️ **Security hardening** - Updated dependencies and security best practices
- 📚 **Documentation overhaul** - Comprehensive guides and API references

### Fixed

- 🐛 Fixed memory leak in development server hot reload
- 🔧 Resolved TypeScript compilation issues with strict mode
- 📦 Fixed package resolution issues with monorepo setups
- 🎨 Corrected CSS import order causing style conflicts

### Breaking Changes

- ⚠️ Updated Node.js requirement to v18+
- ⚠️ Changed default database connection configuration
- ⚠️ Renamed `createApp()` to `createFarmApp()` for clarity
