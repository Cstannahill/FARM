# Community

Join our thriving community of FARM Framework developers worldwide.

## Get Involved

### Discord Community

Join over 2,500 developers in our Discord server for real-time discussions, help, and collaboration.

**Discord Server:** [Join the FARM Discord](https://discord.gg/JmeShQtv)  
**Active Channels:**

- #general - General discussions
- #help - Get help with your projects
- #showcase - Share your creations
- #announcements - Latest updates

### GitHub Community

Contribute to FARM Framework development and stay updated with the latest changes.

**GitHub Repository:** [farm-framework/farm](https://github.com/cstannahill/farm-framework)  
**Ways to contribute:**

- Report bugs and issues
- Submit feature requests
- Contribute code improvements
- Help with documentation

## Community Guidelines

We strive to maintain a welcoming and inclusive environment for all community members.

### Our Values

- **Respect** - Treat everyone with kindness and respect
- **Collaboration** - Help each other grow and learn
- **Innovation** - Share ideas and push boundaries together
- **Inclusivity** - Welcome developers of all backgrounds and skill levels
