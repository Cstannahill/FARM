# CLI Reference

The FARM CLI provides a unified command interface for project creation, development, and deployment. Inspired by Next.js, it supports both quick interactive prompts and detailed non-interactive commands.

<Callout type="info" title="Philosophy">
  Respect both "quick start" developers and "I know exactly what I want"
  developers with intelligent prompting and comprehensive arguments.
</Callout>

## Project Creation

### `farm create`

Create a new FARM project with optional template and features.

<ApiMethod method="CLI" endpoint="farm create <project-name> [options]" />

#### Templates

Choose from pre-built project templates:

<CodeTabs examples={[
  {
    language: "bash",
    label: "Basic Templates",
    code: `# Simple React + FastAPI + MongoDB
farm create my-app --template basic

# Chat app with streaming AI responses

farm create my-app --template ai-chat

# Data dashboard with ML insights

farm create my-app --template ai-dashboard`  },
  {
    language: "bash", 
    label: "Specialized Templates",
    code:`# E-commerce platform with payments
farm create my-app --template ecommerce

# Content management system

farm create my-app --template cms

# FastAPI backend only (no React)

farm create my-app --template api-only

# Custom template from URL/path

farm create my-app --template custom --url https://github.com/user/template`
}
]} />
