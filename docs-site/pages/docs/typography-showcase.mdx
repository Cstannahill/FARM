## Code Examples

Here's an inline code example: `npm install farm-framework`

### JavaScript Example

```javascript
import { createFarmApp } from "farm-framework";

const app = createFarmApp({
  plugins: ["auth", "database"],
  config: {
    port: 3000,
    database: {
      url: process.env.DATABASE_URL,
    },
  },
});

app.listen(() => {
  console.log("Farm app running!");
});
```

### TypeScript Example

```typescript
interface FarmConfig {
  port: number;
  database: {
    url: string;
    pool?: {
      min: number;
      max: number;
    };
  };
  auth?: {
    providers: string[];
    jwt: {
      secret: string;
      expiresIn: string;
    };
  };
}

const config: FarmConfig = {
  port: 3000,
  database: {
    url: process.env.DATABASE_URL!,
    pool: {
      min: 2,
      max: 10,
    },
  },
};
```

### Step-by-Step Instructions

<Step number={1} title="Install the Framework">
First, install the FARM framework using your preferred package manager:

```bash
npm install farm-framework
# or
pnpm add farm-framework
# or
yarn add farm-framework
```

</Step>

<Step number={2} title="Configure Your App">
Create a configuration file to set up your application:

```typescript
// farm.config.ts
export default {
  database: {
    provider: "postgresql",
    url: process.env.DATABASE_URL,
  },
  auth: {
    providers: ["github", "google"],
  },
};
```

</Step>

<Step number={3} title="Start Development">
Run the development server to start building your application:

```bash
npm run dev
```

</Step>

## More Advanced Examples

### Nested Components

<Info title="Advanced Usage">
You can nest other components inside callouts and notes:

```typescript
import { useFarm } from "farm-framework";

export function MyComponent() {
  const { auth, db } = useFarm();

  return (
    <div>
      <h1>Welcome {auth.user?.name}</h1>
    </div>
  );
}
```

<Warning>
Make sure to handle loading states properly when using these hooks.
</Warning>
</Info>

