# Plugin System

The plugin system lets the framework be extended safely across frontend, backend, AI and database layers. Plugins declare metadata, configuration schema and lifecycle hooks.

```text
┌─────────────────────────────────────────────────────────────────┐
│                    FARM Plugin Ecosystem                       │
└─────────────────────────────────────────────────────────────────┘
```

Official plugins include authentication, payments, storage and more, while community plugins can be distributed via a registry.
