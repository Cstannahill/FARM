### Step-by-Step Instructions

<div className="space-y-4">

<Step number="1" title="Project Setup">
Create a new FARM project with the AI template for quick setup with intelligent features built-in.

```bash
farm create my-ai-app --template ai-chat
```

</Step>

<Step number="2" title="Configure AI Provider">
Set up your preferred AI provider in the configuration file. FARM supports multiple providers for flexibility.

```typescript
// farm.config.ts
export default defineConfig({
  ai: {
    provider: "ollama",
    models: ["llama3.2", "codellama"],
    temperature: 0.7,
  },
});
```

</Step>

<Step number="3" title="Implement Features">
Use the generated hooks and components to build your AI-powered features with type safety and error handling.

```typescript
const { generate, loading } = useAI();
const response = await generate("Create a user dashboard");
```

</Step>

</div>

## API Documentation Components

### API Methods

<ApiMethod method="GET" endpoint="/api/users" description="Retrieve all users with pagination">

**Query Parameters:**

- `page` (number): Page number for pagination
- `limit` (number): Items per page (default: 20)
- `search` (string): Search term for filtering

**Response:**

```json
{
  "users": [
    {
      "id": "user_123",
      "name": "John Doe",
      "email": "john@example.com"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150
  }
}
```

</ApiMethod>

<ApiMethod method="POST" endpoint="/api/users" description="Create a new user">

**Request Body:**

```json
{
  "name": "Jane Smith",
  "email": "jane@example.com",
  "role": "user"
}
```

**Response:**

```json
{
  "id": "user_124",
  "name": "Jane Smith",
  "email": "jane@example.com",
  "role": "user",
  "createdAt": "2024-01-15T10:30:00Z"
}
```

</ApiMethod>
